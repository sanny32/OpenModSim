console

Объект console предоставляет доступ к консоли отладки.

Статические методы

console.clear()

Очистить консоль.

console.debug()

Выводит сообщение на консоль с уровнем журнала debug.

console.error()

Выводит сообщение об ошибке.

console.log()

Для общего вывода информации в журнал.

console.warning()

Выводит предупреждающее сообщение.

Date

Дата JavaScript в основном определяется как количество миллисекунд, прошедших с эпохи ECMAScript, которая определяется как полночь в начале 1 января 1970 года по Гринвичу (эквивалентно эпохе UNIX).

Конструктор

Date()

При вызове в качестве функции возвращает строковое представление текущей даты и времени. Все аргументы игнорируются. Результат такой же, как при выполнении new Date().toString().

new Date()

При вызове в качестве конструктора возвращает новый объект Date.

Статические методы

Date.now()

Возвращает числовое значение, соответствующее текущему времени — количество миллисекунд, прошедших с 00:00:00 UTC 1 января 1970 года, без учета дополнительных секунд.

Date.parse()

Анализирует строковое представление даты и возвращает количество миллисекунд с 1 января 1970 года, 00:00:00 UTC, при этом дополнительные секунды игнорируются.

Date.UTC()

Принимает те же параметры, что и самая длинная форма конструктора (т. е. от 2 до 7), и возвращает количество миллисекунд с 1 января 1970 года, 00:00:00 UTC, при этом дополнительные секунды игнорируются.

Возвращает день недели (0 – 6) для указанной даты по местному времени.

Возвращает год (4 цифры для 4-значного года) указанной даты по местному времени.

Date.prototype.getHours()

Возвращает час (0 – 23) указанной даты по местному времени.

Date.prototype.getMilliseconds()

Возвращает миллисекунды (0 – 999) указанной даты по местному времени.

Date.prototype.getMinutes()

Возвращает минуты (0 – 59) указанной даты по местному времени.

Date.prototype.getMonth()

Возвращает месяц (0 – 11) указанной даты по местному времени.

Date.prototype.getSeconds()

Возвращает секунды (0 – 59) указанной даты по местному времени.

Date.prototype.getTime()

Возвращает числовое значение указанной даты в виде количества миллисекунд с 1 января 1970 года, 00:00:00 UTC. (Отрицательные значения возвращаются для предыдущего времени.)

Возвращает смещение часового пояса в минутах для текущей локали.

Date.prototype.getUTCDate()

Возвращает день (число) месяца (1 – 31) в указанную дату по всемирному времени.

Date.prototype.getUTCDay()

Возвращает день недели (0 – 6) в указанную дату по всемирному времени.

Date.prototype.getUTCFullYear()

Возвращает год (4 цифры для 4-значного года) в указанную дату по всемирному времени.

Date.prototype.getUTCHours()

Возвращает часы (0 – 23) указанной даты по всемирному времени.

Date.prototype.getUTCMilliseconds()

Возвращает миллисекунды (0 – 999) указанной даты по всемирному времени.

Date.prototype.getUTCMinutes()

Возвращает минуты (0 – 59) указанной даты по всемирному времени.

Date.prototype.getUTCMonth()

Возвращает месяц (0 – 11) указанной даты по всемирному времени.

Date.prototype.getUTCSeconds()

Возвращает секунды (0 – 59) указанной даты по всемирному времени.

Date.prototype.setDate()

Устанавливает день месяца для указанной даты по местному времени.

Date.prototype.setFullYear()

Устанавливает полный год (например, 4 цифры для 4-значного года) для указанной даты по местному времени.

Date.prototype.setHours()

Устанавливает часы для указанной даты по местному времени.

Date.prototype.setMilliseconds()

Устанавливает миллисекунды для указанной даты по местному времени.

Date.prototype.setMinutes()

Sets the minutes for a specified date according to local time.

Date.prototype.setMonth()

Устанавливает месяц для указанной даты по местному времени.

Date.prototype.setSeconds()

Устанавливает секунды для указанной даты по местному времени.

Date.prototype.setTime()

Задает для объекта Date время, представленное числом миллисекунд с 1 января 1970 года, 00:00:00 UTC. Используйте отрицательные числа для числа, предшествующего времени.

Date.prototype.setUTCDate()

Устанавливает день месяца для указанной даты по всемирному времени.

Date.prototype.setUTCFullYear()

Устанавливает полный год (например, 4 цифры для 4-значного года) для указанной даты по всемирному времени.

Date.prototype.setUTCHours()

Устанавливает час для указанной даты по всемирному времени.

Date.prototype.setUTCMilliseconds()

Устанавливает миллисекунды для указанной даты по всемирному времени.

Date.prototype.setUTCMinutes()

Устанавливает минуты для указанной даты по всемирному времени.

Date.prototype.setUTCMonth()

Устанавливает месяц для указанной даты по всемирному времени.

Date.prototype.setUTCSeconds()

Устанавливает секунды для указанной даты по всемирному времени.

Date.prototype.toDateString()

Возвращает "дату" части Date в виде удобочитаемой строки, например 'Четверг, 12 апреля 2018 года'.

Date.prototype.toISOString()

Преобразует дату в строку, соответствующую расширенному формату ISO 8601.

Возвращает строку, представляющую Дату, используя toISOString().

Date.prototype.toLocaleDateString()

Возвращает строку с представлением части этой даты с учетом местонахождения на основе системных настроек.

Возвращает строку с представлением этой даты с учетом местоположения.

Date.prototype.toLocaleTimeString()

Возвращает строку с представлением временной части этой даты с учетом местоположения на основе системных настроек.

Date.prototype.toString()

Date.prototype.toTimeString()

Возвращает строку, представляющую указанный объект Date.

Math

Math – это встроенный объект со свойствами и методами для математических констант и функций. Это не функциональный объект.

Math работает только с типом Number. Объект Math не работает с типом BigInt.

Описание

В отличие от многих других глобальных объектов, Math не является конструктором. Все свойства и методы Math являются статическими. Вы ссылаетесь на константу pi как Math.PI и вызываете функцию синуса как Math.sin(x), где x — это значение метода. аргумент. Константы определяются с полной точностью действительных чисел в JavaScript.

Статические свойства

Math.E

Число Эйлера и основание натуральных логарифмов; примерно 2,718.

Math.LN2

Натуральный логарифм 2; примерно 0,693.

Math.LN10

Натуральный логарифм 10; примерно 2,303.

Math.LOG2E

Логарифм E по основанию 2; примерно 1,443.

Math.LOG10E

Логарифм E по основанию 10; примерно 0,434.

Math.PI

Отношение длины окружности к ее диаметру; примерно 3,14159.

Math.SQRT1_2

Квадратный корень из ½; примерно 0,707.

Math.SQRT2

Квадратный корень из 2; примерно 1,414.

Статические методы

Math.abs()

Возвращает абсолютное значение x.

Math.acos()

Возвращает арккосинус x.

Math.acosh()

Возвращает гиперболический арккосинус x.

Math.asin()

Возвращает арксинус x.

Math.asinh()

Возвращает гиперболический арксинус числа.

Math.atan()

Возвращает арктангенс x.

Math.atanh()

Возвращает гиперболический арктангенс x.

Math.atan2()

Возвращает арктангенс частного своих аргументов.

Math.cbrt()

Возвращает кубический корень x.

Math.ceil()

Возвращает наименьшее целое число, большее или равное x.

Math.clz32()

Возвращает количество начальных нулевых битов 32-битного целого числа x.

Math.cos()

Возвращает косинус x.

Math.cosh()

Возвращает гиперболический косинус x.

Math.exp()

Возвращает ex, где x — аргумент, а e — число Эйлера (2,718…, основание натурального логарифма).

Math.expm1()

Возвращает вычитание 1 из exp(x).

Math.floor()

Возвращает наибольшее целое число, меньшее или равное x.

Math.fround()

Возвращает ближайшее представление x с плавающей запятой одинарной точности.

Math.hypot()

Возвращает квадратный корень из суммы квадратов своих аргументов.

Math.imul()

Возвращает результат 32-битного целочисленного умножения x и y.

Math.log()

Возвращает натуральный логарифм (loge; также ln) числа x.

Math.log1p()

Возвращает натуральный логарифм (loge; также ln) 1 + x для числа x.

Math.log10()

Возвращает логарифм x по основанию 10.

Math.log2()

Возвращает логарифм x по основанию 2.

Math.max()

Возвращает наибольшее из нуля или более чисел.

Math.min()

Возвращает наименьшее из нуля или более чисел.

Math.pow()

Возвращает основание x в степень степени y (то есть xy).

Math.random()

Возвращает псевдослучайное число между 0 и 1.

Math.round()

Возвращает значение числа x, округленное до ближайшего целого числа.

Math.sign()

Возвращает знак x, указывающий, является ли x положительным, отрицательным или нулевым.

Math.sin()

Возвращает синус x.

Math.sinh()

Возвращает гиперболический синус x.

Math.sqrt()

Возвращает положительный квадратный корень из x.

Math.tan()

Возвращает тангенс x.

Math.tanh()

Возвращает гиперболический тангенс x.

Math.trunc()

Возвращает целую часть x, удаляя все дробные цифры.

Script

Объект Script предоставляет доступ к функциям скрипта.

Статические свойства

Script.runCount

Возвращает количество выполнений скрипта в периодическом режиме.

Script.period

Возвращает количество миллисекунд между запусками скрипта в периодическом режиме.

Статические методы

Script.onInit(functionRef)

Выполняет функцию при первом запуске скрипта.

Script.setTimeout(functionRef, delay)

Метод устанавливает таймер, который выполняет функцию по истечении времени таймера.

Script.stop()

Останавливает выполнения скрипта.

Server

Объект Server обеспечивает доступ к функциям сервера MODBUS.

Статические методы

Server.onChange(Register, address, functionRef)

Выполняет функцию functionRef(value) при изменении значения регистра тип Register с адресом address.

Server.readCoil(address)

Возвращает значение регистра Coil по адресу address.

Server.readDiscrete(address)

Возвращает значение регистра Discrete по адресу address.

Server.readInput(address)

Возвращает значение регистра Input по адресу address.

Server.readHolding(address)

Возвращает значение регистра Holding по адресу address.

Server.readLong(Register, address, swapped)

Возвращает 32-битное целое значение (long) из регистров типа Register с адресами от address до address + 1. Параметр swapped изменяет порядок регистров, когда функция преобразует набор регистров в длинное целое значение (long).

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.readUnsignedLong(Register, address, swapped)

Возвращает беззнаковое 32-битное целое значение (unsigned long) из регистров типа Register с адресами от address до address + 1. Параметр swapped изменяет порядок регистров, когда функция преобразует набор регистров в беззнаковое длинное целое значение (unsigned long).

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.readDouble(Register, address, swapped)

Возвращает значение двойной точности (double) из регистров типа Register c адресами от address до address + 3. Параметр swapped изменяет порядок регистров, когда функция преобразует набор регистров в значение двойной точночти (double).

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.readFloat(Register, address, swapped)

Возвращает значение с плавающей запятой (float) из регистров типа Register с адресами от address до address + 1. Параметр swapped изменяет порядок регистров, когда функция преобразует набор регистров в значение с плавающей запятой (float).

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.writeCoil(address, value)

Записывает value в регистр Coil по адресу address.

Server.writeDiscrete(address, value)

Записывает value в регистр Discrete по адресу address.

Server.writeInput(address, value)

Записывает value в регистр Input по адресу address.

Server.writeHolding(address, value)

Записывает value в регистр Holding по адресу address.

Server.writeLong(Register, address, value, swapped)

Записывает 32-битное целое число (long) value в адреса регистров типа Register. Параметр swapped изменяет порядок регистров, когда функция преобразует длинное целое значение (long) в набор регистров.

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.writeUnsignedLong(Register, address, value, swapped)

Записывает 32-битное беззнаковое целое число (unsigned long) value в адреса регистров типа Register. Параметр swapped изменяет порядок регистров, когда функция преобразует беззнаковое длинное целое значение (unsigned long) в набор регистров.

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.writeDouble(Register, address, value, swapped)

Записывает значение value двойной точночности (double) в адреса регистров типа Register. Параметр swapped изменяет порядок регистров, когда функция преобразует значение двойной точности (double) в набор регистров.

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Server.writeFloat(Register, address, value, swapped)

Записывает число с плавающей запятой (float) value в адреса регистров типа Register. Параметр swapped изменяет порядок регистров, когда функция преобразует значение с плавающей запятой (float) в набор регистров.

Примечание. Эта функция доступна только для регистров типа Register.Input и Register.Holding.

Storage

Storage обеспечивает доступ к локальному хранилищу скриптов. Это позволяет, например, добавлять, изменять или удалять сохраненные элементы данных.

Объект Storage хранит элементы во время выполнения скрипта (однократно или периодически). При запуске скрипта в объекте Storage нет элементов.

Статические свойства

Storage.length

Возвращает целое число, представляющее количество элементов данных, хранящихся в объекте Storage.

Статические методы

Storage.key(index)

Вернуть имя n-го ключа по index в хранилище.

Storage.getItem(key)

При передаче имени ключа key будет возвращено значение этого ключа.

Storage.setItem(key, value)

При передаче имени ключа key и значения этот ключ будет добавлен в хранилище или обновлен value этого ключа, если он уже существует.

Storage.removeItem(key)

При передаче имени ключа key этот ключ будет удален из хранилища.

Storage.clear()

При вызове все ключи удаляются из хранилища.
